HTML: The Language of the Web

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It defines the structure of a webpage and allows you to organize content with headings, paragraphs, images, and more.

Example:

<!DOCTYPE html>
<html>
<head>
    <title>My First HTML</title>
</head>
<body>
    <h1>Hello, HTML!</h1>
    <p>This is a paragraph.</p>
</body>
</html>

CSS: Styling the Web

CSS (Cascading Style Sheets) allows you to style your HTML elements. It helps control colors, layouts, fonts, and more.

Example:

body {
    background-color: lightblue;
}
h1 {
    color: navy;
    text-align: center;
}
